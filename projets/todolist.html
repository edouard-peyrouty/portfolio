Todo List

Application web développée en Python avec le framework Flask, permettant de gérer une liste de tâches. L'application propose la création, modification et suppression de tâches, un système de filtrage par statut, ainsi qu'une corbeille permettant de restaurer les tâches supprimées avant de les effacer définitivement.

Gestion et filtrage des tâches

L'application permet de créer des tâches avec un titre et une description, de les modifier, de les marquer comme terminées et de les filtrer selon leur statut : toutes, en cours ou terminées. Chaque tâche est supprimable individuellement depuis la liste.

Corbeille et restauration

Les tâches supprimées sont déplacées dans une corbeille plutôt qu'effacées immédiatement, offrant la possibilité de les restaurer en cas de suppression accidentelle. Un bouton permet de vider définitivement la corbeille en une seule action.

Architecture et base de données

L'application suit une architecture MVC : le fichier principal définit les routes Flask qui délèguent chaque action à un contrôleur central gérant la logique métier. Ce contrôleur s'appuie sur deux repositories distincts — un pour les tâches actives, un pour les tâches supprimées — qui accèdent chacun à leur table dans une base de données SQL locale. Les données sont ensuite transmises aux templates Jinja2 pour le rendu des vues.
