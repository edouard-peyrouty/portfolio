Application web de gestion de base de données

Application web développée en C# avec le framework Blazor (.NET) permettant de gérer les produits et les entreprises d'une base de données. L'application offre une interface interactive pour consulter, ajouter, modifier et supprimer les entrées.

Gestion des produits

La vue Produits affiche l'ensemble des articles enregistrés en base, avec leurs attributs (gencode, libellé, prix, statut, date d'ajout) et l'entreprise associée. Chaque ligne propose des actions de modification et de suppression, ainsi qu'un bouton d'ajout global pour créer de nouveaux produits.

Gestion des entreprises

La vue Companies liste les entreprises avec le nombre de produits rattachés à chacune. Depuis la page d'édition d'une entreprise, il est possible de modifier son libellé et de gérer directement ses produits associés, offrant ainsi une navigation centralisée entre les deux entités.

Architecture

L'application suit une architecture MVC adaptée à Blazor, avec une séparation claire des responsabilités : les modèles définissent les entités métier, les DTOs assurent le transfert des données entre les couches, et les services encapsulent la logique d'accès à la base de données. Les composants Blazor jouent le rôle de vues et de contrôleurs, en s'appuyant sur l'injection de dépendances native de .NET.
