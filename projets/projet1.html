Application web de visualisation météorologique

Application web dynamique développée en PHP permettant de générer des courbes et des cartes météorologiques à partir de données issues de l’API SYNOP de Météo-France. L’application suit une architecture MVC, avec une base de données SQL pour la gestion des données statiques (comptes utilisateurs) et des requêtes à l'API pour les données météo.

Cartes et courbes

L’application permet aux utilisateurs de choisir les données météorologiques à visualiser et de les restituer sous forme de courbes ou de cartes météorologiques.

Systèmes de comptes

L’application intègre un système complet de gestion des comptes utilisateurs et des sessions, incluant l’inscription, la connexion et la déconnexion. L’accès aux fonctionnalités est conditionné à une validation préalable des comptes par un administrateur, qui dispose également de la possibilité de gérer et supprimer les comptes si nécessaire.

Architecture

L’application suit une architecture MVC (Modèle-Vue-Contrôleur) avec un premier modèle dédié à la base de données SQL gérant les données utilisateurs et un second chargé des requêtes à l’API météo.
